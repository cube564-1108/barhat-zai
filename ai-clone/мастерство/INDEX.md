## Как пополнять

1. Выбери область компетенции
2. Найди 3-5 признанных авторов
3. По каждому автору выдели 1-3 гениальных метода
4. Создай файл: `<область>/<имя-автора>.md`
5. Обнови INDEX.md

---

## Правила

- Каждый файл — 3-5 KB концентрата
- Метод = название + суть + шаги + когда применять
- Связывай методы через [[вики-ссылки]]
- Писать на русском
