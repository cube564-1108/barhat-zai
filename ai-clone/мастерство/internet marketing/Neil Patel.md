## Метод 1: The Hub-and-Spoke Content Strategy

**Суть:** Создай один "hub" контент (pillow page) и множество "spokes" (related articles), которые all link back to hub. Это усиливает SEO для целевой keyword.

**Структура:**
- **Hub Page** — comprehensive guide на target keyword (3000+ words)
- **Spoke Articles** — shorter articles на related keywords (500-1000 words)
- **Internal Links** — every spoke links to hub
- **External Links** — hub attracts external links

**Когда применять:**
- Для competitive keywords
- При построении topical authority
- Для increasing organic traffic

**Шаги внедрения:**
1. Выбери target keyword (например, "интернет-маркетинг для малого бизнеса")
2. Создай comprehensive Hub Page
3. Сгенерируй список related keywords
4. Напиши 10-30 spoke articles
5. Link каждый spoke к hub
6. Продвижай hub для external links

## Метод 2: The skyscraper Technique 2.0

**Суть:** Найди контент конкурента, который rank на target keyword → создай something better → outreach тем, кто linked на оригинал.

**Версия 2.0:**
- Не просто "больше" — но structurally better
- Добавь interactive элементы, original data, case studies
- Outreach не только тем, кто linked — но и тем, кто shares similar content

**Когда применять:**
- Для quick wins в competitive niches
- Для acquisition of high-quality backlinks
- Когда недостаточно organic growth

**Шаги внедрения:**
1. Найди competitor content с many backlinks
2. Создай superior version (add value, design, data)
3. Найди всех, кто linked на оригинал (Ahrefs, SEMrush)
4. Outreach с personalised emails
5. Follow up 2-3 раза

## Метод 3: The SEO Audit Framework

**Суть:** Systematic approach к SEO-аудиту без инструментов. Focus на pillars: technical, content, backlinks.

**3 Pillars SEO Audit:**
1. **Technical SEO** — site speed, mobile-friendliness, indexability
2. **Content Audit** — keyword gaps, thin content, duplicate content
3. **Backlink Audit** — toxic links, anchor text diversity, link velocity

**Когда применять:**
- При падении organic traffic
- Перед запуском новой кампании
- Для competitive analysis

**Шаги внедрения:**
1. Прогони сайт через PageSpeed Insights
2. Проверь mobile usability через Google Search Console
3. Аудируй top 10 pages — keyword density, internal links
4. Проверь backlinks через Ahrefs/SEMrush
5. Приоритизуй fixes по impact
