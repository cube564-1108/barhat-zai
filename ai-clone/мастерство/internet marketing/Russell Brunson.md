# Russell Brunson

## Кто автор
Основатель ClickFunnels, построил 100+ миллионный бизнес на sales funnels. Автор "DotCom Secrets", "Expert Secrets", "Traffic Secrets". Превратил сложный маркетинг в простые фреймворки, которые используют миллионы предпринимателей.

## Метод 1: Perfect Webinar Framework

**Суть:** Скрипт вебинара, который конвертирует холодную аудиторию в покупателей. Структура ~90 минут, разбитых на 4 части.

**Структура:**
1. **The Origin Story (15 мин)** — личная история, почему ты создаёшь продукт
2. **The One Thing (30 мин)** — обучение 3 секретов, где секрет #3 — твой продукт
3. **The Stack (15 мин)** — презентация оффера с бонусами
4. **The Q&A (30 мин)** — работа с возражениями через ответы

**Когда применять:**
- Для запуска info-продуктов или high-ticket услуг
- Для warming холодной аудитории
- Для продаж без sales team

**Шаги внедрения:**
1. Напиши Origin Story — path от проблемы к решению
2. Выбери 3 секрета — секрет #1 и #2 дают value, секрет #3 = твой продукт
3. Создай Stack — ценность + цена + бонусы
4. Отрепетируй тайминг
5. Запиши и протестируй

## Метод 2: The "Who, Not How" Framework

**Суть:** Не пытайся делать всё сам. Найди "кого" — партнёров, аффилиатов, каналы трафика. Это mindset для масштабирования.

**Принцип:**
- Твой job — создать продукт и фреймворк
- Job других — продавать и доставлять
- Ты расплачиваешься процентом, не фиксом

**Когда применять:**
- На этапе масштабирования
- Когда не хватает ресурсов на all-in-one маркетинг
- Для партнёрских запусков

**Шаги внедрения:**
1. Выбери один канал (например, аффилиаты)
2. Создай материалы для партнёров
3. Предложи attractive комиссию
4. Автоматизируй выплату

## Метод 3: The 2-Comma Club Framework

**Суть:** Формула для создания 7-фигурного воронка. Focus не на продукте, а на offer.

**Формула:**
```
Offer = Product + Bonuses + Scarcity + Urgency + Guarantee
```

**Когда применять:**
- При создании нового продукта
- Когда текущий offer не конвертирует
- Для differentiation от конкурентов

**Шаги внедрения:**
1. Усиль свой продукт бонусами
2. Добавь scarcity (ограниченное количество)
3. Добавь urgency (deadline)
4. Создай сильную гарантию
5. Тестируй и оптимизируй
