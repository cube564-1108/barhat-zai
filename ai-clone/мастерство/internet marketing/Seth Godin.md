# Seth Godin

## Кто автор
Маркетинговый визионер, автор 21 бестселлера. Создатель концепций "permission marketing" и "purple cow". Не учит "как продать больше", а "как быть remarkable".

---

## Метод 1: Permission Marketing

**Суть:** Маркетинг с разрешения, а не прерывание. Люди дают permission → ты sent им value → они покупают.

**Классический marketing vs Permission Marketing:**
- **Interruption Marketing** — TV ads, cold calls, spam = люди игнорируют
- **Permission Marketing** — email, подписка, relationship = люди хотят получать

**5 ступеней Permission Marketing:**
1. **Strangers** → привлечь внимание
2. **Friends** → дай что-то free, спроси permission
3. **Lovers** → sendrelevant, personalised content
4. **Superfans** → они покупают и рекомендуют
5. **Former Customers** → winning back

**Когда применять:**
- При построении email-листа
- Для long-term relationship с клиентами
- Когда interruption marketing не работает

**Шаги внедрения:**
1. Создай valuable Lead Magnet
2. Испрашивай permission (opt-in)
3. Отправляй исключительно relevant контент
4. Продавай soft, не навязчиво
5. Уважай permission (unsubscribe option)

## Метод 2: Purple Cow

**Суть:** В saturated markets нужно быть remarkable — как фиолетовая корова на лугу обычных коров. Усиль的产品, не marketing.

**Принципы Purple Cow:**
- **Safe is risky** — обычное = invisibly
- **Remarkable = worth talking about** — люди говорят о тебе
- **Niche first** — создай для небольшой группы early adopters
- **Design for sneezers** — создай продукт, который spread word-of-mouth

**Когда применять:**
- При запуске нового продукта
- Когда реклама не даёт ROI
- Для differentiation от конкурентов

**Шаги внедрения:**
1. Найди свою "фиолетовую корову" — что делает тебя remarkable?
2. Создай продукт, который worth talking about
3. Тестируй на early adopters
4. Позволь word-of-mouth spread
5. Не надdoing remarkable — remarkable = niche

## Метод 3: The Dip

**Суть:** Quit или stick? Knowing когда quitting — это smart move, а не failure.

**The Dip — это:**
- Long valley между началом и mastery
- Место, где大多数人 quit
- Но именно через Dip path к success

**Правила:**
- Quit projects with no long-term potential
- Stick через Dip для projects с потенциалом
- Strategic quitting frees resources для better things

**Когда применять:**
- При принятии решения о pivot
- Для evaluation текущих initiatives
- Для фокусировки на "right things"

**Шаги внедрения:**
1. Оцени свои текущие projects — какие в Dip?
2. Определи: есть ли light в конце tunnel?
3. Quit тех, где нет потенциала
4. Allocate ресурсы на прохождение Dip для promising projects
5. Stick через Dip — это path к exceptional results
