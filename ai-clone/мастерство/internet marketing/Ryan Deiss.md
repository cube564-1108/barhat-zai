## Метод 1: Customer Value Journey (CVJ)

**Суть:** 8-ступенчатый путь клиента от незнакомца до промоутера. Это альтернатива классической воронке, которая учитывает long-term ценность клиента.

**8 ступеней:**
1. **Aware** (Осведомлённость) — клиент узнаёт о тебе
2. **Subscribe** (Подписка) — оставляет email/телефон
3. **Convert** (Конверсия) — первая покупка
4. **Excite** (Воодушевление) — positives первые впечатления
5. **Ascend** (Восхождение) — повторные покупки, upsell
6. **Advocate** (Адвокат) — рекомендует другим
7. **Promote** (Промоутер) — активно продвигает бренд
8. **Bond** (Привязанность) — эмоциональная связь с брендом

**Когда применять:**
- При проектировании маркетинговой воронки
- Когда есть трафик, но нет повторных продаж
- Для поиска дыр в customer journey

**Шаги внедрения:**
1. Нарисуй 8 ступеней для своего бизнеса
2. Отметь, где сейчас клиенты "падают"
3. Создай кампанию для каждого перехода между ступенями
4. Измеряй конверсию на каждом этапе

## Метод 2: Customer Value Optimization (CVO)

**Суть:** Математический подход к маркетингу. Фокус не на стоимости лида, а на ценности клиента.

**Формула CVO:**
```
Profit = (Lead Volume × Lead-to-Customer Rate × Average Cart Value × Transactions per Customer × Margin) − Cost
```

**Когда применять:**
- Когда платный трафик становится дорогим
- Когда нужно принять решение о масштабировании
- Для сравнения эффективности каналов

**Шаги внедрения:**
1. Собери данные по формуле
2. Найди "bottleneck" — weakest звено
3. Улучши weakest звено на 10-20%
4. Повторяй цикл

## Метод 3: Lead Magnet Tripwire Offer Core Offer

**Суть:** Структура оффера для максимальной конверсии. Последовательность: бесплатный лид-магнит → низкий ticket трипвайер → основной продукт → high-ticket backend.

**Структура:**
- **Lead Magnet** — бесплатный, high value, low friction (PDF, видео, чек-лист)
- **Tripwire Offer** — 7-47$, no-brainer, impulse buy
- **Core Offer** — основной продукт, 97-497$
- **High-Ticket** — премиум, 997+

**Когда применять:**
- При запуске нового продукта
- Когда текущая воронка даёт низкий ROI
- Для увеличения LTV клиента

**Шаги внедрения:**
1. Создай Lead Magnet, который решает одну конкретную проблему
2. Сделай Tripwire — "no-brainer" оффер за 7-47$
3. Продай Core Offer через tripwire buyers
4. Добавь High-Ticket backend
