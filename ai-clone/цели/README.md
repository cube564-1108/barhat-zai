# Цели

> Твои цели — краткосрочные, среднесрочные, долгосрочные.

## Основные цели в компании Бархат

- Построение устойчивого бизнеса
- Постоянное увеличение прибыли компании
- Оптимизация и автоматизация всех процессов в компании
- Повышение лояльности клиентов к компании
