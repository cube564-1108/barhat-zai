## Добавить новую область

**Промпт:**
```
"Я работаю над задачей в области [ниша].
Найди 3-5 авторов с доказанными методологиями.
По каждому: кто + 1-3 метода с алгоритмом «что → как → когда».
Формат: mastery/[область]/[автор].md
Создай mastery/[область]/INDEX.md с картой авторов."
```

**Структура области:**
```
mastery/
└── [область]/
    ├── INDEX.md          ← карта авторов области
    ├── [автор-1].md      ← кто + методы
    └── [автор-2].md
```
