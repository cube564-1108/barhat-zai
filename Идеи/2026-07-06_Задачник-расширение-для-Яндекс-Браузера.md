# Задачник — расширение для Яндекс Браузера

**Дата создания:** 2026-07-06

## Суть идеи

Расширение-задачник, которое позволяет быстро создавать задачи из любого контекста в браузере — особенно в Яндекс Мессенджере, но и на любых сайтах.

## Как работает

1. Пользователь выделяет текст на странице (например, в переписке)
2. Правый клик → контекстное меню «Добавить в задачу»
3. Задача сохраняется в расширении
4. Popup показывает список всех задач с возможностью:
   - Отметить выполненной
   - Удалить
   - (опционально) редактировать

## Технический стек

- **Платформа:** Chromium Extension (Chrome, Яндекс Браузер)
- **Хранилище:** chrome.storage.local (или chrome.storage.sync для синхронизации)
- **Компоненты:**
  - manifest.json — описание расширения
  - background.js — обработка контекстного меню
  - popup.html + popup.js — интерфейс задачника
  - content.js — (опционально) альтернативные способы захвата

## Почему это может быть полезно Бархату

В переписке с менеджерами, курьерами, клиентами часто всплывает "сделать потом". Сейчас это либо теряется, либо уходит в заметки/телефон. Расширение позволит одним кликом фиксировать задачу прямо из контекста переписки.
