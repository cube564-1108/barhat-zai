# Chat Quality Monitor — система оценки качества переписки менеджеров

**Дата создания:** 2026-07-06

## Суть идеи

Автоматическая оценка качества переписки менеджеров с клиентами в чате (RetailCRM) с дашбордом для анализа.

## Метрики оценки

1. **Скорость реакции** — время от сообщения клиента до ответа менеджера
2. **Правильность общения** — наличие приветствия, представления, вежливости, прощания
3. **Орфографические ошибки** — количество ошибок на 1000 символов
4. **Профессионализм продажника** — эвристики (выявление потребности, предложение решения, работа с возражениями, призыв к действию)

## Как работает

1. **Историческая выгрузка** — скрипт выгружает 3 месяца переписок из RetailCRM
2. **Анализ эвристик** — Python-модуль анализирует каждую переписку по метрикам
3. **База данных** — PostgreSQL хранит результаты анализа
4. **Webhook** — RetailCRM отправляет уведомления о новых сообщениях, система анализирует их в real-time
5. **Дашборд** — Streamlit показывает графики и фильтры по менеджерам, периодам, городам

## Технический стек

- **Python 3.11+** — основной язык
- **FastAPI** — webhook-сервер
- **PostgreSQL** — база данных
- **Streamlit** — дашборд
- **pyspellchecker / language-tool-python** — проверка орфографии
- **RetailCRM API** — источник данных

## Почему это может быть полезно Бархату

- **Контроль качества** — объективная оценка работы менеджеров, а не "на глаз"
- **Обучение** — видно конкретные проблемы (медленная реакция, нет вежливости, орфография)
- **Тренды** — динамика по времени, сравнение менеджеров между собой
- **ROI** — можно связать качество переписки с конверсией в заказ
