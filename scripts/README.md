# Скрипты для сверки заказов

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Выгрузка из RetailCRM

```bash
python export_retailcrm.py --from "2026-07-01 00:00:00" --to "2026-07-07 23:59:59" --output retailcrm_orders.json
```

### Параметры:

- `--from` — начальная дата и время (обязательно)
- `--to` — конечная дата и время (опционально)
- `--output` — имя выходного файла (по умолчанию `retailcrm_orders.json`)

### Примеры:

```bash
# За один день
python export_retailcrm.py --from "2026-07-07 00:00:00" --to "2026-07-07 23:59:59"

# За месяц
python export_retailcrm.py --from "2026-07-01 00:00:00" --to "2026-07-31 23:59:59"

# С указанного времени по сегодня
python export_retailcrm.py --from "2026-07-01 00:00:00"
```

### Выходной формат:

Каждый заказ содержит:
- `id` — ID заказа
- `number` — номер заказа
- `status` — код статуса
- `status_name` — название статуса
- `sum` — сумма товаров
- `total_sum` — итоговая сумма
- `created_at` — дата создания
- `updated_at` — дата обновления
- `customer_id` — ID клиента
- `customer_name` — имя клиента
- `customer_phone` — телефон клиента
- `payment_type` — способ оплаты
- `delivery_type` — способ доставки
- `source` — источник данных (`retailcrm`)

## Выгрузка из МойСклад

```bash
python export_moysklad.py --from "2026-07-01 00:00:00" --to "2026-07-07 23:59:59" --output moysklad_orders.json
```

### Параметры:

- `--from` — начальная дата и время (обязательно)
- `--to` — конечная дата и время (опционально)
- `--output` — имя выходного файла (по умолчанию `moysklad_orders.json`)

### Примеры:

```bash
# За один день
python export_moysklad.py --from "2026-07-07 00:00:00" --to "2026-07-07 23:59:59"

# За месяц
python export_moysklad.py --from "2026-07-01 00:00:00" --to "2026-07-31 23:59:59"

# С указанного времени по сегодня
python export_moysklad.py --from "2026-07-01 00:00:00"
```

### Выходной формат (МойСклад):

Каждый заказ содержит:
- `id` — UUID заказа в МойСклад
- `name` — номер заказа
- `status` — ID статуса
- `status_name` — название статуса
- `sum` — сумма заказа
- `total_sum` — итоговая сумма
- `created_at` — дата создания
- `updated_at` — дата обновления
- `moment` — момент документа (дата заказа)
- `customer_id` — ID контрагента
- `customer_name` — название контрагента
- `description` — описание (может содержать ID из RetailCRM)
- `external_code` — внешний код (для связи с внешними системами)
- `applicable` — проведен ли заказ
- `source` — источник данных (`moysklad`)

## Запуск обеих выгрузок сразу

```bash
# Выгрузка из обеих систем за один период
python export_retailcrm.py --from "2026-07-01 00:00:00" --to "2026-07-07 23:59:59"
python export_moysklad.py --from "2026-07-01 00:00:00" --to "2026-07-07 23:59:59"
```

## Сверка заказов

```bash
python reconcile_orders.py --crm retailcrm_orders.json --ms moysklad_orders.json --output report.json
```

### Параметры:

- `--crm` — файл с заказами из CRM (обязательно)
- `--ms` — файл с заказами МойСклад (обязательно)
- `--output` — файл для сохранения отчета (по умолчанию `reconciliation_report.json`)

### Логика сверки:

**Что проверяется:**
1. ✅ Заказы создаются только в CRM (источник всех заказов)
2. ✅ В МойСклад подгружаются только заказы со статусами: "выполнен", "отменен", "удержание"
3. ✅ Поиск соответствия заказов по:
   - externalCode (ID заказа CRM)
   - description (содержит ID заказа CRM)
   - номеру заказа
4. ✅ Сравнение статусов
5. ✅ Сравнение суммы (допустимая погрешность 1 рубль)
6. ✅ Поиск заказов в МойСклад, которые там не должны быть

### Результаты сверки:

Отчет содержит:
- **Общая статистика** — количество заказов в каждой системе
- **Полностью совпадающие** — заказы без расхождений
- **С расхождениями** — заказы с разными статусами или суммами
- **Не найденные** — заказы из CRM, которых нет в МойСклад
- **Неожиданные** — заказы в МойСклад со статусами, которые не должны быть там

### Пример использования:

```bash
# Полный цикл: выгрузка + сверка
python export_retailcrm.py --from "2026-07-01 00:00:00" --to "2026-07-07 23:59:59"
python export_moysklad.py --from "2026-07-01 00:00:00" --to "2026-07-07 23:59:59"
python reconcile_orders.py --crm retailcrm_orders.json --ms moysklad_orders.json
```
