# Tasks — инструкция

## Зачем

При ошибке `context window limit` история чата теряется. Этот каталог — спасение: важное о задаче хранится в файлах, а не в истории диалога.

## Как использовать

### Начало задачи

1. Скопируй `template.md` → `tasks/имя-задачи.md`
2. Заполни "Описание" и "Следующие шаги"
3. В чате: `Работаем по плану tasks/имя-задачи.md`

### В процессе работы

После важного этапа:
- Открой `tasks/имя-задачи.md`
- Добавь в "Сделано" то, что завершено
- Обнови "Следующие шаги"
- Запиши важные решения в "Заметки"

### При ошибке контекста

1. Новый чат
2. `Прочитай tasks/имя-задачи.md и продолжи с текущего статуса`
3. Работаем дальше

## Пример

См. `tasks/example.md` — заполненный пример.

## Правила

- Одна задача = один файл
- Название файла: `краткое-описание.md` или `YYYY-MM-DD-название.md`
- Обновляй файл регулярно, после важных этапов
- В "Скопировать для нового чата" — минимум, чтобы быстро восстановить контекст
