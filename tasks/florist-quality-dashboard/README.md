# Florist Quality Dashboard

Дашборд для анализа качества работы флористов и салонов.

## 📁 Файлы

- `process_quality_data_full.py` — **Итоговый рабочий скрипт** для генерации HTML дашборда из CSV
- `florist-quality-dashboard.html` — Готовый HTML дашборд
- `source_data.csv` — Исходные данные из Pyrus

## 🚀 Запуск

```bash
python process_quality_data_full.py
```

После запуска создастся/обновится `florist-quality-dashboard.html`.

## ✅ Выполненные задачи

### 1. Удалено "/18" из карточки флориста
**Было:** `Флорист 6: 14.2/18 (6 зак.)`
**Стало:** `Флорист 6: 14.2 (6 зак.)`

### 2. Исправлена логика закрытых салонов
При выборе периода скрываются карточки салонов, у которых нет заказов в выбранный период.

### 3. Добавлена статистика по категориям
В каждой карточке салона теперь отображается статистика по категориям товаров:
- Клубничный букет
- Цветочный букет
- Клубнично-цветочный букет
- И др.

### 4. Удалены insights из карточек флористов
Убран блок с "✅" и "⚠️" для более чистого вида.

### 5. Добавлено динамическое обновление карточек
При выборе периода обновляются:
- Средний рейтинг салона
- Количество заказов
- Статистика по категориям
- Список флористов

## 📊 Структура дашборда

- **KPI карточки** — общая статистика за выбранный период
- **Рейтинг салонов** — график топ-15 салонов
- **Карточки салонов** — детальная информация по каждому салону:
  - Средний рейтинг
  - Количество заказов
  - Статистика по категориям
  - Список флористов
- **Проблемные задачи** — задачи с низким баллом

## 🔧 Технические детали

- **Python 3.11+**
- **Chart.js** для графиков
- **Статический HTML** — не требует сервера
- **JavaScript фильтрация** по периодам
