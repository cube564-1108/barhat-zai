# Business — второй мозг

Карта бизнес-контекста. Всё, что нужно знать о компании, продукте, клиенте, деньгах.

## Правило работы

**Читай только нужный файл. Не загружай всё сразу.**

Открывай конкретную папку под задачу:
- Нужна инфа о продукте → `products/`
- Надо понять клиента → `audience/`
- Считаешь экономику → `economics/`
- Планируешь маркетинг → `marketing/`
- Ставишь цели → `goals/`
- Готовишь контент → `assets/`

## Структура

| Папка | Что внутри |
|-------|------------|
| [`products/`](products/) | Что продаём: ассортимент, тарифы, цены, позиционирование |
| [`audience/`](audience/) | Кто покупает: аватары, сегменты, боли, путь клиента |
| [`economics/`](economics/) | Деньги: выручка, расходы, юнит-экономика, прогноз |
| [`marketing/`](marketing/) | Каналы: воронка, конкуренты, контент-стратегия |
| [`goals/`](goals/) | Цели: годовые, квартальные, месячные, KPI |
| [`assets/`](assets/) | Бренд: гайдлайны, лого, отзывы, копи-банк |
| [`life-metrics.md`](life-metrics.md) | Как брать свежие цифры из базы (не хардкодить) |
| [`raw/`](raw/) | Исходники: транскрипты зумов, переписки, черновики |

## Как пополнять

Каждая папка → заполняй через интервью. Не выдумывай — спрашивай у владельца.
