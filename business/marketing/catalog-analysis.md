## ✹ Сильные стороны

1. **Навигация по "Кому/Событие"** — клиенту не надо думать, что выбрать
2. **ТОП‑бейджи** ("Выбрали 1300+ раз") — социальное доказательство
3. **Фильтры по цвету/форме/составу** — быстрый подбор нужного
4. **Комбо-наборы** — upsell (цветы + сладость)
5. **Акции видны** — бесплатная доставка, -10% на самовывоз
6. **Фото готового букета** — гарантия соответствия

## ⚠️ Точки роста

| Проблема | Решение | Приоритет |
|----------|---------|-----------|
| Много похожих товаров (гортензия 8+ позиций) | Объединить в карточку с выбором размера | Средний |
| Нет быстрого "Добавить в корзину" на карточке | Добавить быструю кнопку | Высокий |
| Нет рекомендаций "С этим покупают" | Кросс-селл блок | Высокий |
| Нет сортировки по цене/популярности | Добавить фильтры | Средний |
| Не заполнены цены на гелиевые шары | Добавить диапазон | Низкий |
