# 2026-07-20: Обновление отчета качества + кнопка "Обновить"

1. Обновить данные по качеству сборки букетов за июль 2026

- Запустил для обновления из Pyrus API

- Flask проксирует запросы к Pyrus API (токен хранится только на сервере)

- Запушен в GitHub
