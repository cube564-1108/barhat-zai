# Менеджер паролей - 2025-07-10

## Задача
Создать менеджер паролей на HTML с хранением данных в файле на компьютере.

## Как решал
1. Проверил существующие версии файлов - нашёл несколько черновиков
2. Выбрал самую продвинутую версию как основу
3. Добавил кнопки "Показать пароль" и "Копировать"
4. Оптимизировал отображение длинных ссылок
5. Исправил проблему с колонкой "Раздел"
6. Внедрил File System Access API для хранения данных в файле
7. Организовал файлы в отдельную папку

## Результат
**Да** - полностью рабочий менеджер паролей с:
- Мастер-паролем и шифрованием
- Хранением данных в JSON файле на диске
- Кнопками показа/скрытия паролей
- Копированием паролей и ссылок
- Категориями/разделами
- Поиском по записям

Файл: `password-manager/password-manager.html`

## Что можно было лучше
1. Сразу создать структуру папок, а не работать в корне
2. Использовать систему контроля версий для черновиков
3. Продумать файловое хранилище с начала, а не добавлять потом
