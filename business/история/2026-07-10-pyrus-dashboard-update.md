## Как это работает

```
Pyrus API → generate_dashboard_from_pyrus.py → process_quality_data_full.py → HTML Dashboard
```

1. **Pyrus API:** Выгрузка данных из формы #1327961
2. **generate_dashboard_from_pyrus.py:** Авторизация, получение заявок, маппинг полей
3. **process_quality_data_full.py:** Расчёт статистики, генерация HTML
4. **HTML Dashboard:** Интерактивный дашборд с фильтрацией и графиками

## Найденные и исправленные проблемы

### Проблема 1: Неправильный парсинг полей Pyrus
**Было:** Скрипт использовал `task.get('values', [])`, но Pyrus возвращает `fields`
**Стало:** Исправлено на `task.get('fields', [])`

### Проблема 2: Неправильное извлечение multiple_choice
**Было:** `value['choice_value']` - такого поля нет
**Стало:** `value['choice_names'][0]` - правильный путь к значению

### Проблема 3: Обработка даты
**Было:** Только формат "2026-07-10T06:29:11Z"
**Стало:** Поддержка обоих форматов - с временем и без
