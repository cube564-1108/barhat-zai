# Разработка дашборда качества работы флористов

**Дата:** 2026-07-10
**Задача:** Создать интерактивный дашборд для анализа качества работы флористов

## Что было сделано

### 1. Генерация HTML дашборда
- ✅ Создан `process_quality_data_full.py` для генерации HTML с полной фильтрацией
- ✅ Дашборд обрабатывает 14,524 заказов по 5 городам
- ✅ Полный пересчёт статистики при выборе периода (Вариант B)

### 2. Функционал дашборда
- ✅ Фильтрация по 24 периодам (месяцам)
- ✅ Кнопка обновления данных
- ✅ График среднего рейтинга салонов (топ-15)
- ✅ Подписи данных на графике
- ✅ Две вкладки: "За всё время" и "За 6 месяцев"
- ✅ Детальный график салона по клику (динамика за 6 месяцев)
- ✅ Округление значений до десятых
- ✅ Динамические заголовки с выбранным периодом

### 3. Визуализация
- 🎨 Градиентный дизайн в стиле бренда
- 📊 Гистограмма рейтинга салонов
- 📈 Линейный график динамики качества
- 🎯 Цветовая кодировка: зелёный (≥14), жёлтый (12-13.9), красный (<12)

### 4. Технические решения
- Chart.js + ChartDataLabels plugin
- Vanilla JavaScript без зависимостей
- Модальное окно для детального графика
- Автоматическая инициализация при загрузке

## Результат

**Файл:** `florist-quality-dashboard.html` (5.4MB)

**Функции:**
- `calculateStats()` — пересчёт статистики
- `updateUI()` — обновление интерфейса
- `applyPeriodFilter()` — фильтрация по периоду
- `openSalonModal()` — детальный график салона
- `switchTab()` — переключение вкладок
