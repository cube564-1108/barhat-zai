# Сессия: Настройка Amvera и GitHub

## Задача
Настроить автоматический деплой статического сайта на Amvera через GitHub.

## Что делал
1. Проверил конфигурацию [amvera.yaml](amvera.yaml) — была некорректной
2. Создал правильный [Dockerfile](Dockerfile) для nginx:alpine
3. Настроил GitHub как внешний репозиторий для Amvera
4. Потратил много времени на отладку 404

## Результат
✅ Сайт работает: `https://barhat-zai.cube564.amvera.io`

## Что можно было лучше
**Критическая ошибка:** Не проверил, создан ли домен в Amvera. Потратил полдня на отладку 404, когда проблема была в отсутствии домена.

**Правило на будущее:** Перед отладкой деплоя проверить ВСЕ базовые шаги:
- Проект создан
- Файлы загружены
- **Домен создан и активен** ← КРИТИЧНО!
- Сборка выполнена
