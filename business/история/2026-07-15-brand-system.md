# Брендинговая система БАРХАТ

**Задача:** Привести отчёт "Качество сборки букетов" к фирменному стилю БАРХАТ и сделать стиль переиспользуемым.

**Как решал:**
1. Создал папку `brand/` с двумя файлами:
   - `tokens.css` — CSS переменные цветов, градиентов и шрифтов
   - `brand.css` — базовые стили для всех отчётов
2. Модифицировал Python скрипт генерации HTML для подключения brand стилей
3. Заменил фиолетовый градиент на фирменный винно-розовый
4. Обновил Dockerfile для копирования brand файлов в nginx

**Результат:** Да
- Создана единая точка_truth для бренда
- Главный отчёт обновлён в фирменном стиле
- Все будущие отчёты подключают те же файлы

**Что можно было лучше:**
- Сначала создал отдельную страницу `/florist-quality.html`, хотя нужно было обновить главную `/index.html`
- Потерял время на ненужное копирование файла

**Было / Стало:**
- Было: хардкод цветов (#667eea, #764ba2) в каждом HTML
- Стало: CSS переменные `var(--barkhat-*)` в едином tokens.css
- Было: фиолетовый градиент
- Стало: винный фон #411330 + розовый градиент БАРХАТ
- Было: Segoe UI
- Стало: Vollkorn (заголовки) + Inter (текст)
