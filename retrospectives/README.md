# Retrospectives — Рефлексии после сессий

Рефлексия после каждой важной сессии. Помогает извлечь уроки, зафиксировать что-то новое, не повторять ошибки.

## Формат файла

`YYYY-MM-DD_HH-MM_краткое-описание.md`

Например:
- `2026-07-04_09-30_audit-funnel.md`
- `2026-07-04_14-15_price-analysis.md`

## Когда создавать

После каждой сессии, где:
- Была решена важная задача
- Была совершена ошибка
- Был принят важный решения
- Был получен новый инсайт

## Как заполнять

Используй шаблон `TEMPLATE.md`. Копируй его, переименуй под текущую дату и заполни.

## Зачем

1. **Фиксировать уроки** — через месяц забудешь
2. **Не повторять ошибки** — отвергнутые подходы записаны
3. **Видеть прогресс** — как было → как стало
4. **Развивать clone** — новые правила в `ai-clone/feedback/`
