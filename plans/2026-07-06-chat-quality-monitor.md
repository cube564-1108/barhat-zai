# 2026-07-06: Chat Quality Monitor — система оценки качества переписки менеджеров

## Описание

Система автоматической оценки качества переписки менеджеров с клиентами в чате (RetailCRM) с дашбордом для анализа.

**Метрики оценки:**
- Скорость реакции (время от сообщения клиента до ответа)
- Правильность общения (приветствие, представление, вежливость, прощание)
- Орфографические ошибки
- Профессионализм продажника (эвристики)

**Режим работы:** авто-обновление через webhook от RetailCRM

### Фаза 3. Модуль анализа эвристик [ ]

**Цель:** реализовать анализ без AI (скорость, орфография, маркеры).

**Deliverable:**
- Python-модуль `chat_analyzer.py`
- Каждая переписка получает числовую оценку по метрикам

**Задачи:**
- [ ] Метрика "Скорость реакции" — время между сообщениями
- [ ] Метрика "Орфография" — pyspellchecker/language-tool
- [ ] Метрика "Правильность общения" — поиск фраз-маркеров
  - Приветствие (привет, здравствуйте, добрый день)
  - Представление (меня зовут, это...)
  - Вежливость (пожалуйста, спасибо, извините)
  - Прощание (до свидания, хорошего дня, всего доброго)
- [ ] Метрика "Профессионализм" — эвристики
  - Выявление потребности (что вас интересует, какой букет, для кого)
  - Предложение решения (могу предложить, вот хороший вариант)
  - Работа с возражениями (понимаю, давайте обсудим)
  - Призыв к действию (оформить заказ, перейти к оформлению)

**Критерий "сделано":**
Модуль на вход принимает переписку, на выходе — структурированная оценка с баллами.

## Приложения

### А. Стек технологий

- **Python 3.11+** — основной язык
- **FastAPI** — webhook-сервер
- **PostgreSQL** — база данных
- **Streamlit** — дашборд
- **pyspellchecker / language-tool-python** — орфография
- **RetailCRM API** — источник данных

### Б. Бюджет

- **Hosting:** нужен сервер для FastAPI + БД (можно TimeWeb/Beget/Selectel ~500 руб/мес)
- **AI:** не нужен (эвристики)

### В. Риски

- RetailCRM API может иметь лимиты по частоте запросов
- Объём данных за 3 месяца может быть большим
- Webhook может терять сообщения при сбоях

### Г. Будущие улучшения

- AI-оценка профессионализма через Claude API
- Мобильное приложение для менеджеров
- Интеграция с Telegram для уведомлений
- А/B тестирование промптов для оценки
