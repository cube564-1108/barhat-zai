## API эндпоинты (финальный список)

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/sales/current-month` | Текущий месяц (с 1 числа) |
| GET | `/api/sales/current-month?year=2026&month=7` | Произвольный месяц |
| GET | `/api/sales/compare-periods?from=...&to=...` | Сравнение периодов |
| GET | `/api/sales/monthly-comparison?year=2026` | По месяцам года |
| POST | `/api/sales/cache/clear` | Очистить кэш |
