# План: Экран очереди заказов для флористов

**Дата создания:** 2026-07-08
**Статус:** `[ ]`

## Описание фичи

Веб-страница для флористов с очередью заказов, которые переданы флористу (статус `send-to-florist`). Флорист видит:
- Список заказов
- Описание заказа (клиент, адрес доставки, время)
- Фото букета из каталога
- Возможность изменить статус заказа

Возможна сортировка по дате: сегодня, завтра, следующие дни.

## Технический стек

- **Backend:** Python Flask (уже используется в проекте)
- **Frontend:** HTML + JavaScript (plain)
- **API:** RetailCRM API v5

## Примечания

- **API:**
  - Заказы: `/api/v5/orders?filter=send-to-florist`
  - Каталог: `/api/v5/store/products`
  - Фото: `product.imageUrl` или `offer.images[]`
- **Статусы для перехода:**
  - `send-to-florist` → `at-work`
  - `at-work` → `complete`
- **Кеширование:** Каталог меняется редко, можно кешировать на 1 час
