# План: Аналитика по продажам — Бархат

**Дата:** 2026-07-17
**Статус:** Черновой план

## Задача

Создать раздел «Аналитика» в главном дашборде с данными по продажам из RetailCRM:
1. Сумма отгрузки за текущий месяц в разрезе по салонам
2. Сравнение периода текущего месяца (1-17 число) с тем же периодом прошлого года
3. Статистика по закрытым месяцам текущего года vs прошлый год

## Спецификация данных из RetailCRM

### Фильтр заказов
- **Статусы:** Все, КРОМЕ:
  - `Отменен`
  - `Заказ не доставлен`

### Поля заказов для выгрузки
```python
url += "&fields=id,number,status,createdAt,customer"
url += "&fields=sum,totalSum,cost,orderMethod"
```

### Привязка к салону
- **Поле:** `order.store` или `customFields` — нужно проверить через API

- **Поле:** `sum` или `totalSum` — "Сумма по товарам/услугам"
