# План: Дашборд флориста

> **Дата создания:** 2026-07-15
> **Статус:** Черновой план (фаза 1)
> **Цель:** Создать дашборд для флористов с интеграцией в RetailCRM

## Краткое описание

Дашборд для флористов — веб-приложение для планшета/смартфона, которое показывает очередь заказов в реальном времени. Флористы видят заказы, принимают в работу, собирают, фотографируют готовый букет — всё уходит в RetailCRM.

```
RetailCRM (webhook) → Node.js Backend (WebSocket) → Планшет (PWA)
```

## Технический стек

```
Backend:  Node.js + Fastify + WebSocket
Frontend: React + Vite (PWA)
Realtime: WebSocket (библиотека `ws`)
Кеш:     In-memory (без БД для начала)
Хостинг:  Amvera
```
