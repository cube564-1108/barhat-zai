## Краткое описание проекта

Дашборд для флористов — веб-приложение для планшета/смартфона, которое показывает очередь заказов в реальном времени. Флористы видят заказы, принимают в работу, собирают, фотографируют готовый букет — всё уходит в RetailCRM.

```
RetailCRM (webhook) → Node.js Backend (WebSocket) → Планшет (PWA)
```

## Ключевые требования (изнес)

```
Масштаб:     10 салонов → 10 дашбордов
Устройства:   Планшет (стена/стол) + смартфон
Интеграция:   RetailCRM напрямую
Фильтр:       По салону
Вкладки:      "На сегодня" / "На завтра"
Сортировка:   По "времени готовности для флориста"
Уведомления:  Звук + визуальный сигнал
```

│  Товары в заказе:                            │
│  [Букет "Любимый"]  │  [Состав]              │
